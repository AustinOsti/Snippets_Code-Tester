** Code Snippets for Testing Code **
To use:
1. Create your test JS code and save with apppropriate title (e.g. 1_Object_Literals.js).
2. In the main.html file, point the <script> tag to your JS file. 

To view results:
1. run nodemon server.js to start the node server
2. open a browser window & type in localhost:3000
3. open the developer tools to view results of the test data.
